# HRP2025-quantitative_analysis
# Data Analysis for: 

This repository contains the R code used for the quantitative data analysis and figure generation in the manuscript **""**.

## Contents
* `[Rscriptfor_GIThub.R]`: The primary R script containing the code for data preparation, calculation of external calibration curves via regression analysis for the absolute quantification of nucleosides measured by LC-MS/MS, and data visualization.

## Requirements
To execute the script, **R** (R Core Team, 2025) is required. The use of the RStudio IDE is highly recommended for running the script.

Additionally, the following R packages must be installed:
* `readxl`
* `writexl`

## Data Availability
Due to confidentiality agreements with the involved companies, the raw data used in this study cannot be made publicly available. The primary focus of this research is on establishing methodological comparability rather than comparing the specific performance of different instruments. The analysis script is provided here to ensure transparency and allow readers to retrace the methodological steps and statistical procedures applied.

## Usage
To review or run the code (with alternative data), download the script and open it in RStudio. Ensure that your working directory is set to the file location (`Session -> Set Working Directory -> To Source File Location`) before executing the code.
